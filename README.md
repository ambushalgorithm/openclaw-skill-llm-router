# OpenClaw Skill LLM Router

Universal brain skill for OpenClaw that routes LLM calls via capability-aware model selection (v2), then dispatches to the appropriate LLM CLI backend (Claude, OpenAI, Ollama, or others). Features 41 models, tier-based classification, cost-optimized routing, and unified usage tracking.
